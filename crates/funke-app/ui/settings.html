Commands

Which built-in providers answer your searches.

Web search engine
Where “Search the web for …” sends you.
Vault: unlock with Windows Hello
After one master-password unlock, later unlocks are a Hello prompt. Stores a session key protected by Windows — tradeoff in SECURITY.md.
Vault: website icons
Show site favicons on vault entries, fetched from your Bitwarden server's icon service (it learns the sites you search).
Vault: suggest for the focused app
When you summon Funke over Discord (or a GitHub tab), offer that credential in the empty overlay — matched by process, window title, and the browser's address bar. Off means vault entries only ever appear behind v.
Vault: autotype presses Enter
After typing the password, submit the form automatically. Turn off to type the credentials and stop, leaving Enter to you. Applies to the built-in sequence only — a custom one below is typed exactly as written.
Vault: autotype sequence
What autotype types, if not the usual username ⇥ password. Tokens: {USERNAME} {PASSWORD} {TOTP} {TAB} {ENTER} {DELAY=500}. A single entry can override this with an autotype custom field in Bitwarden.
Vault: lock on screen lock
Lock the vault immediately when you lock Windows (Win+L).
Vault: auto-lock after idle
Lock the vault after this long without using it.
File index folders
What file search covers. Empty means your home folder; changes re-index within seconds.

Snippets

Text you paste often — a signature, an address, a block of boilerplate. Find one with s, and Enter types it into the window you came from.

Placeholders
Resolved when the snippet is pasted, not when it is saved: {DATE}, {TIME}, {DATETIME} (or your own format, {DATE:%d.%m.%Y}), {CLIPBOARD} for what you last copied, and {CURSOR} for where the caret should land. Anything else in braces is typed exactly as written.
